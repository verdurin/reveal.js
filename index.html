What is eMedLab?

Hardware and Software

252 hypervisors, each with 512GB RAM and 2 x 12-core Intel Ivy Bridge 2.40GHz CPUs

6,048 cores and 129TB RAM

Some of those cores and some of that RAM will be used by the operating system and hence not available for users

4.3PB storage in /emed/bulk

1.2PB in /emed/scratch

Software

Red Hat OpenStack Platform 5

OpenStack Icehouse release

eMedLab "Service Philosophy"

Project-based

Access allocated via project applications

Access managed on the infrastructure via OpenStack projects

Each project creates VMs customised for their specific workflows

In the future, custom VM images may be shared between projects

eMedLab resources are not on the public internet by default

Access mediated by Cisco VPN

This is how to manage VMs and connect to them
